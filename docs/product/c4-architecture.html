Level 3 — Component Diagram (Backend Gateway)
AI Service
Gemini Vision Analyzer Gemini
Receives Firecrawl screenshot + markdown. Extracts industry, layout structure, color palette, content blocks as structured JSON.
Gemini Template Matcher Gemini
Receives source site analysis + ALL template screenshots in one multimodal call. Scores visual similarity, ranks top 3.
Input: ~50 images + analysis JSON
Context: 1M+ tokens
GPT Content Generator OpenAI
Takes the matched template's content_slots + user brief. Generates copy for every section: headlines, paragraphs, CTAs.
GPT Chat Interpreter OpenAI
Parses natural language edit requests into structured action JSON. Maintains conversation context per editing session.
Whisper STT OpenAI
Transcribes voice audio chunks from WebSocket stream. Supports streaming for real-time feedback.
TTS Engine OpenAI
Converts AI question text to speech audio. Streamed back to client for voice mode Q&A.
Crawler Service
Firecrawl Client Firecrawl
Wraps Firecrawl API. Calls /scrape for single pages, /crawl for multi-page. Returns markdown, metadata, screenshots.
POST api.firecrawl.dev/v1/scrape
POST api.firecrawl.dev/v1/crawl
Crawl Cache
Redis-backed cache. Same URL within 1 hour returns cached results. Prevents duplicate API spend.
Site Builder Service
Template Catalog
In-memory catalog of all WP templates with metadata (industries, sections, content_slots) + screenshot URLs. Loaded at startup.
InstaWP Provisioner InstaWP
Creates WP site from template via InstaWP API. Polls until ready. Maps custom domain. Returns site credentials.
POST app.instawp.io/api/v2/sites
Plugin Content Pusher WAAS Plugin
Pushes AI-generated content to site via WAAS Controller Plugin endpoints. Creates pages, sections, uploads media.
POST {site}/wp-json/waas/v1/pages
PATCH {site}/wp-json/waas/v1/sections/:id
POST {site}/wp-json/waas/v1/media
WP Editor Service (Chat)
Action Planner
Receives parsed edit intent from GPT. Validates against site's current state. Maps to WAAS Plugin endpoint calls.
Plugin Executor WAAS Plugin
Calls WAAS Controller Plugin endpoints on the live site. Snapshots before edits. Reports progress. Supports rollback.
PATCH {site}/wp-json/waas/v1/sections/:id
POST {site}/wp-json/waas/v1/snapshot
POST {site}/wp-json/waas/v1/snapshot/restore
Site State Cache
Fetched via GET /waas/v1/site-map. Cached structure of pages, sections, fields. Refreshed on each edit.
GET {site}/wp-json/waas/v1/site-map
Domain Service
Namecheap Client Namecheap
Domain availability check, registration, nameserver configuration. XML API with whitelisted IP auth.
Cloudflare Client Cloudflare
Zone creation, A record setup, security config (HTTPS, DDoS), performance (Brotli, HTTP/3).
Billing Service
Stripe Client Stripe
Checkout sessions, subscription management, webhook handler, customer portal links.
Usage Meter
Tracks AI token usage (both providers), sites created, domains registered. Enforces plan limits.
