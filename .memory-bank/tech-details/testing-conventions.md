# Testing Conventions

## Test Types

- **Unit tests**: Test individual functions/components
- **Integration tests**: Test module interactions
- **E2E tests**: Test full user flows

## Running Tests

No test framework is currently configured in the project. Consider adding:

```bash
# Suggested setup
npm install --save-dev jest  # or vitest for ES modules
```

## Coverage Requirements

- Not yet defined
- Recommended minimum: 80% for critical paths (AI service, domain workflow, onboarding)
