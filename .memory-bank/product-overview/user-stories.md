# User Stories

## Format

Each story follows: **As a [role], I want [feature], so that [benefit].**

## Stories

### Website Creation

- As a user, I want to describe my website via voice commands, so that I can create a site without technical knowledge.
- As a user, I want AI to generate my website content, so that I have professional copy without hiring a writer.
- As a user, I want to go through a guided onboarding wizard, so that my website is set up correctly from the start.

### Website Migration

- As a user, I want to migrate my existing website, so that I can benefit from AEO/GEO optimization without starting over.
- As a user, I want the system to scrape my current site, so that all my content is preserved during migration.

### Domain Management

- As a user, I want to register a domain through the platform, so that I don't need to use a separate service.
- As a user, I want automatic DNS and SSL setup, so that my site is secure and properly configured.

### Content Editing

- As a user, I want to edit my site via a light chat-based editor, so that I can make changes conversationally.
